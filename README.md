# Challenge Alura Store

## Descripción del Proyecto

Análisis completo del rendimiento de 4 tiendas del señor Juan para determinar la mejor oportunidad de inversión y la venta de una de ellas. Este proyecto utiliza técnicas de data science para evaluar múltiples métricas de negocio y proporcionar recomendaciones basadas en datos.

## Objetivos

- **Identificar** la tienda con mejor y menor rendimiento integral
- **Analizar** patrones de ventas y comportamiento del cliente
- **Proporcionar** recomendaciones para la venta de tienda para futuros proyectos e inversiones

## Métricas Analizadas

| Categoría | Métricas Específicas |
|-----------|---------------------|
| **Financieras** | Ingresos totales, Rentabilidad por producto |
| **Ventas** | Productos más/menos vendidos, Categorías con mayor rendimiento |
| **Clientes** | Calificaciones promedio, Satisfacción |
| **Operaciones** | Costos de envío, Eficiencia logística |
